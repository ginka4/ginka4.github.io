Linux文件的操作

发表于 2025-12-30 更新于 2026-01-06 分类于 从零开始的Linux生活 阅读次数： 本文字数： 1.3k 阅读时长 ≈ 2 分钟

　　本文章主要讲解了入门Linux系统的基本文件操作方法，供读者入门参考。

文件的创建

1.cat>file1

直接用键盘写文件内容，结束输入用ctrl+d

重定向符号 > ，管道pipe

2.echo hello>test2

默认情况下，会有一个\n写入文件

echo -n hello>test2

当echo使用了-n选项之后，可以去掉\n换行符

3.touch file3

4.vim file4

硬链接

1.ln命令

初始情况下，file1的初始链接数为1

创建file1的硬链接file2，之后发现硬链接数变成了2

file1于file2是同一个文件

软链接

1.ln -s test1 stest1

作用类似于Windows中的快捷方式

文件的权限修改

1.文字设定法

chmod u+x,g+x ycm.py

2.数字设定法

chmod 640 ycm.py

文件的拷贝(复制)

1.cp命令

cp ycm.py ./test/dir1/

复制文件的同时 ，修改文件的名字

cp ycm.py ./test/dir1/file3.py

2.拷贝文件夹(目录) 加-r,递归执行

在同一个父目录复制文件夹，同时还修改了文件夹的名字

cp -r test test2

在不同目录下

cp -r test2 ~/

3.跨主机传输文件

scp 源文件的位置 目的主机的位置

​ user@host:/path

然后输入目的主机的相应用户的密码

4.移动文件

mv(move) 源文件 目的地

mv ycm.py test/dir2/

5.重命名功能

对同一目录下的文件进行重命名

mv test2 test3

文件的删除

1.rm test4

删除单独的文件

2.rm ~/test2/test3

删除不同的目录下的文件

3.目录的删除

rm test3 -r -r为递归删除

rm -r ~/test2

文件的查找

1.find ~/ -name stdio.h

find 查找位置 按文件名 文件名

在根目录下查找时，前面加sudo提升权限

2.grep 要过滤的关键字 指定要查找的位置 -rn

grep string ./ -rn

-r表示递归查找 -n表示显示关键字在文件中的行号

打包和压缩

1.tar命令

只打包，没有压缩的效果

tar -cvf day02.tar day02/

压缩的效果

tar -zcvf day02.tar.gz day02

拆包和解压缩

1.只拆包，没有解压缩

tar -xvf day02.tar

解压缩

tar -zxvf day02.tar.gz

2.解压Windows生成的*.zip的压缩文件

unzip Linux.zip

查看磁盘空间

1.df -h 查看磁盘

2.free -h 查看内存

对一个命令设置别名

第一步，找到用户家目录下的.bashrc文件

第二步，在该文件的末尾加上新的一行，使用alias命令起别名

第三步，重新加载该文件

$ source ~/.bashrc
